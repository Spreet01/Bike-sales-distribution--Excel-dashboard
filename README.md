Bike Sales Distribution Dashboard (Excel)

Project Overview
This project presents an interactive Bike Sales Distribution Dashboard built using Microsoft Excel to analyze sales patterns, regional distribution, and customer behavior.
The dashboard helps in understanding how bike sales vary across different regions and customer segments.

Tools & Skills Used
* Microsoft Excel
* Pivot Tables
* Data Cleaning
* Data Visualization
* Dashboard Design

Dataset Information
The dataset includes:
* Customer demographics
* Sales transactions
* Regional data
* Product details

 Dashboard Features

* Region-wise sales distribution
* Customer segmentation
* Sales trends
* Revenue analysis
* Category-wise performance

Dashboard Preview

Key Insights
* Identified regions with highest bike sales
* Analyzed customer demographics influencing purchases
* Observed trends in product demand
* Highlighted top-performing segments
